Cómo agregar Google AdSense a Nuxt.js 3 usando el módulo @nuxtjs/google-adsense

por , 12-Mar-2023

Google AdSense es una plataforma de publicidad en línea que permite a los editores monetizar su contenido a través de anuncios en sus sitios web. En este tutorial, aprenderás cómo agregar Google AdSense a un proyecto de Nuxt.js 3 usando el módulo @nuxtjs/google-adsense. Paso 1: Instalar el módulo @nuxtjs/google-adsense

Para agregar Google AdSense a tu proyecto de Nuxt.js 3, primero debes instalar el módulo @nuxtjs/google-adsense. Puedes hacer esto usando npm o yarn.

npm install @nuxtjs/google-adsense

o


yarn add @nuxtjs/google-adsense

Paso 2: Configurar el módulo @nuxtjs/google-adsense

Para configurar el módulo @nuxtjs/google-adsense, agrega lo siguiente a tu archivo nuxt.config.js:


export default {
  modules: [
    '@nuxtjs/google-adsense',
  ],
  googleAdsense: {
    id: 'TU_ID_DE_ADSENSE_AQUÍ',
    pageLevelAds: true,
  },
}

Reemplaza TU_ID_DE_ADSENSE_AQUÍ con tu identificación de AdSense. Paso 3: Agregar anuncios a tus páginas

Una vez que hayas configurado el módulo @nuxtjs/google-adsense, puedes agregar anuncios a tus páginas utilizando la etiqueta .


<template>
  <div>
    <h1>Mi página de inicio</h1>
    <client-only>
      <ins class="adsbygoogle"
           style="display:block"
           data-ad-client="TU_ID_DE_ADSENSE_AQUÍ"
           data-ad-slot="TU_SLOT_DE_ADSENSE_AQUÍ"
           data-ad-format="auto"
           data-full-width-responsive="true"></ins>
    </client-only>
  </div>
</template>

Reemplaza TU_SLOT_DE_ADSENSE_AQUÍ con el slot de AdSense donde deseas que se muestren los anuncios. Paso 4: Verifica que se muestren los anuncios

Una vez que hayas agregado la etiqueta con el código de AdSense a tus páginas, asegúrate de que los anuncios se muestren correctamente en tu sitio web.

¡Eso es todo! Ahora deberías tener anuncios de Google AdSense en tu proyecto de Nuxt.js 3 usando el módulo @nuxtjs/google-adsense.
