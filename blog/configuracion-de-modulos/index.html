Cómo configurar los módulos

En Nuxtjs 3, la condiguración de los módulos en nuxt.config.js se pone dentro del mismo listado de los módulos y no, aparte.

export default {
  modules: [
    // Simple usage
    '@nuxtjs/axios',

    // With options
    ['@nuxtjs/axios', {
      baseURL: 'https://api.example.com'
    }]
  ]
}
