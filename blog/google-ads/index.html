Cómo Agregar Etiquetas de Google Ads al Encabezado de una Aplicación Nuxt.js 3

por Jorge, 24-Mar-2023

En Nuxt.js 3, puedes agregar las etiquetas de Google Ads al encabezado de tu aplicación editando el archivo nuxt.config.js. Para hacer esto, agrega el siguiente código en la sección head:

export default {
  head: {
    script: [
      {
        src: 'https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js',
        'data-ad-client': 'TU-CÓDIGO-DE-CLIENTE-DE-GOOGLE-ADSENSE'
      }
    ]
  }
}

Reemplaza TU-CÓDIGO-DE-CLIENTE-DE-GOOGLE-ADSENSE con el código de cliente que obtuviste de Google AdSense.

Recuerda que para que los anuncios se muestren correctamente, debes seguir las políticas de Google AdSense y asegurarte de que tu sitio cumpla con los requisitos de calidad y contenido.
